# Development

### Installation

1. `git clone git@github.com:lhchanh/Wonolo.git && cd Wonolo`
2. `bundle install`
3. `rake db:create db:migrate`

# Running tests

* `bundle exec rspec`

# Deploy to Heroku:

`git push heroku master`
